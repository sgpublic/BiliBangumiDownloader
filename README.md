## 如何使用

1. 在b站中找到你想看的番，点击追番。
2. 打开哔哩番剧，登录之后即可看到你的追番列表。
3. 找到你追番列表中想缓存的番剧，点击打开详情界面，~~根据需要下载对应剧集即可（下载功能维护中）~~，可直接播放。
